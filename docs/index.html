# XLSX Made easy

This vue Library is a vue wrapper for the excellent: sheet-js The aim of this library is to provide an easy to use, well documented way to parse and create spreadsheets in vue.

WARNING

This is a 0.x.x release, although the components are fully functional we did not have extensive 'field test'.
